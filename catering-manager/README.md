# VostokEda

Приложение для автоматизации организации питания на мероприятиях

## Описание

Приложение разработано для АО "ТехноХолдинг 'Восток'" для автоматизации процессов организации питания на мероприятиях. Система позволяет управлять номенклатурой блюд и напитков, поставщиками, мероприятиями и заказами, контролировать бюджет и формировать отчеты.

## Основные функции

1. **Управление категориями затрат**
   - Создание, редактирование, удаление категорий
   - Гибкая система категоризации расходов

2. **Управление номенклатурой**
   - Ведение справочника блюд, напитков и услуг
   - Привязка к категориям затрат
   - Указание единиц измерения

3. **Управление поставщиками**
   - Ведение базы поставщиков
   - История цен поставщиков
   - Контактная информация и рейтинги

4. **Управление мероприятиями**
   - Создание и планирование мероприятий
   - Установка бюджета
   - Учет количества гостей

5. **Создание заказов**
   - Формирование заказов на питание
   - Контроль бюджета в реальном времени
   - Проверка сроков заказа (не позднее суток до мероприятия)

6. **Отчетность**
   - Отчеты по расходам на мероприятия
   - Визуализация данных (диаграммы)
   - Экспорт в Excel

## Технические требования

- Python 3.8+
- SQLite 3
- Современный компьютер с ОС Windows/Linux/macOS
