# Директория для изображений

Здесь должны находиться изображения для приложения:

1. `logo.png` - логотип приложения (рекомендуемый размер: 200x200 пикселей)
2. `icons/` - директория для иконок:
   - `add.png` - иконка добавления
   - `edit.png` - иконка редактирования
   - `delete.png` - иконка удаления
   - `save.png` - иконка сохранения
   - `export.png` - иконка экспорта
   - `report.png` - иконка отчета
   - `settings.png` - иконка настроек

Примечание: Для работы приложения достаточно пустой директории или только логотипа.
